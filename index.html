Radio

Its is used when only one option to be selected out of several available options. Radio button is presented as small circle on the screen. It is used when you want to limit the users choice to just one option from the range provided.


Checkbox

Checkbox allow one or many options to be selected. Checkbox is presented as small square box on the screen. It is used when you want to allow user to select multiple options.
